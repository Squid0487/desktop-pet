# desktop-pet
A desktop pet program for windows, which you can upload pictures to make your own desktop pet. 

Introduction: 
There are 2 modes of the pet, one is when it is being dragged, and one is not.
When you let go of it in mid-air, it will drop to the bottom itself. 
Use curser to drag the pet around and Esc for quitting the program. 
Use the default pet to try out first!

Download: 
Put all the downloaded files and folders into the same folder. 
The program works best when it's inside the c drive. 
Libaries used: pygame, pywin32, PIL
Install promts (with pip): 
"pip install pygame", 
"pip install pywin32", 
"pip install Pillow" 
Version of python: 3.13.9 
Run the main.py in the terminal to start the program. 
Uploaded files can only be in JPEG PNG and GIF. 
Type file.file_type in the terminal to upload the file e.g. oiiai_stand.png 
Uploaded image should have empty spaces as fewer as possible. 
Behaves just like a window, although it's always on top, you have to click it twice to drag it when you are moving from another window.
